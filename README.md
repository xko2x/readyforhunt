## readyforhunt
tool install all tools need for pentesting on vps and add small alias 

# tools:
* ffuf  
* dirsearch  
* autoSubTakeover  
* subbrute  
* JSParser  
* knockpy  
* lazys3  
* recon_profile  
* sqlmap-dev  
* Sublist3r  
* teh_s3_bucketeers  
* virtual-host-discovery  
* wpscan  
* webscreenshot  
* Massdns  
* Asnlookup  
* Unfurl  
* Waybackurls  
* Httprobe  
* Seclists collection  
* some wordlists  


## Installation
git clone https://github.com/xko2x/readyforhunt  
cd readyforhunt  
chmod +x install.sh  
./install.sh
